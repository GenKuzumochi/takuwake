# Takuwake


## How to use

data.tsv

```
PL番号 卓1への希望度 卓2への希望度 卓3への希望度 ...
```

希望度は大きいほど希望の度合いが大きい。

```
$ python3 solve.py 卓1への希望度 卓2への希望度 卓3への希望度 ...
```

## example

PL1は「卓1:5、卓2:3、卓3:2」と回答

```

2	0	0	10
3	1	6	3
```
